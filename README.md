# ScholarFlow - 文獻探討助手 📚

一個智能的文獻探討助手，幫助研究者快速分析參考文獻並生成高品質的文獻探討初稿。

## ✨ 主要功能

- 🔑 **多組 API Key 管理** - 支援 9 組 Gemini API keys 自動輪替
- 📊 **即時狀態監控** - 視覺化顯示每組 API key 的使用狀態（🟢🟡🔴⚫）
- 📄 **頁碼範圍選擇** - 可指定 PDF 的特定頁碼進行分析
- 🎯 **小節聚焦分析** - 針對特定文獻探討小節進行深度分析
- 🔄 **自動流量管理** - 智能處理 API 流量限制，自動切換 keys
- 📈 **進度即時顯示** - 清楚顯示處理進度和當前狀態

## 🚀 快速開始

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
npm run dev
```

### 建置並部署到 GitHub Pages
```bash
npm run deploy
```

## 📖 使用說明

1. **設定 API Keys** - 輸入多組 Gemini API keys（建議 9 組）
2. **輸入研究細節** - 填寫論文題目、研究重點和當前撰寫小節
3. **上傳參考文獻** - 選擇最多 5 個英文 PDF 論文
4. **上傳範例**（選填）- 可指定頁碼範圍的學長姐範例
5. **開始分析** - 觀察進度和 API key 狀態
6. **查看結果** - 獲得文獻分析和初稿

## 🛠️ 技術棧

- React 18 + TypeScript
- Vite 4
- Google Gemini API
- GitHub Pages

## ⚠️ 注意事項

- 需要有效的 Gemini API keys
- 建議準備多組 API keys 以確保服務不中斷
- AI 生成的內容僅供參考，使用前務必核對原始文獻
